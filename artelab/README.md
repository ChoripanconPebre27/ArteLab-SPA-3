# Artelab SPA
Artelab quiere posicionarse como una tienda digital de materiales artísticos accesibles y de calidad, que promueve a artistas emergentes mediante exhibiciones online. Su desafío es crear un sistema modular para ventas y promociones, asegurando trazabilidad de pedidos y fomentando la creatividad como motor social en comunidades con menos acceso cultural.

Entidades:
  - Producto (material de arte)
  - Categoría
  - Cliente
  - Pedido
  - Promoción

Spring Boot 4.0.5

Java 25
